Description

A tool with a log viewer GUI, written in C++ and built on Qt framework.

I developed it while I was working for Samsung WPC project in IBM ISC Slovakia, for internal purposes only. I can't go into too much detail, but basically this tool evaluates and submits forms in a CMS. In our line of work, that meant a big time saving for everyone in the team, since there is a considerable amount of repetitive tasks being automated on a daily basis.

Under the hood, the tool uses Qt WebKit and its implementation of WebView API. To communicate with the page in WebView, a JavaScript bridge is established. Cool stuff.
